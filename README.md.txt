# Adventure Game 

## Description

The text-based adventure game depicts a story where you, the player appear to have woken up in a mysterious cave, in front of a split path..
Choose wisely.

## Installation

In order to play the game, python v3 is required.
Once Python has been installed on your device, click the adventure_game.py file to run.
